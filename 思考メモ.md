# 1290. Convert Binary Number in a Linked List to Integer
# Step1
 
以下リンク先頭図のようなものを作成して連結させることをイメージする  
ポインタ部を辿ってデータ部にたどり着くらしい  
cf. https://qiita.com/tsudaryo1715/items/12c4848028716ab015bb  

これに似たデータ構造として配列があるが、これは辿るのではなく指定の添字に直接アクセスするものらしい  

どちらが優れているのかという点については  
データアクセス  
　・配列  
　　　添字を指定すればすぐアクセスできるので  
データの追加削除  
　・リスト  
　　　ポインタで繋ぎ変えがすぐできるので  

方針は、ノードがある限り先頭から一つずつ進めて行って、その値を足す、ということまで思いついた  
しかし、返り値をどうすればいいのかが分からなくなったのでAIにヒントを聞いたり、自分で調べて  （https://zenn.dev/komiya_atsushi/articles/a42a675d5273b5c54150）  
以下のような自然言語化に至った  
1.リストの先頭から末尾に向かってノードがある限り一つずつ進めていく  
2.返り値問題はここ？  
普通は以下のように考える  
1 * 2^2 + 0 * 2^1 + 1 * 2^0 = 5  

しかし、AIによれば、数を左から順に作るイメージにシフトしろとのこと  
つまり普通にずらして入れてを繰り返せばよい  
1101を対象にするなら以下のようになる  
result = 0  
1. result = 0×2 + 1 （0をずらして、1を入れて）=1  
2.result = 1×2 + 1（1.の結果を次の位に押し出して、1を入れて）=3  
3. result = 3×2 + 0（2.の結果を次の位に押し出して、0を入れて）=6  
4. result = 6×2 + 1（3.の結果を次の位に押し出して、1を入れて）=13  

これを踏まえると以下のようになる  
```
class Solution:
    def getDecimalValue(self, head: Optional[ListNode]) -> int:
        result = 0
        node = head
        while node is not None:
            result = result * 2 + node.val
            node = node.next

        return result
```

# Step3  
以下のようにシフト演算子と論理和を用いた  
命名の方はさすがにanswerはこの問題限定といった感じで汎用的ではないので避けた  
ただresultも中々読み手に意味が伝わらないと思うので...  
まず合計なのでsum系にする  
(cf:https://github.com/Nbotter/leetcode-easy-swe-practice/pull/10#discussion_r2936168216)  
sum_binary_digitなんかいい気がする  
これらのことから以下のようなコードが導かれる    
```
class Solution:
    def getDecimalValue(self, head: Optional[ListNode]) -> int:
        sum_binary_digit = 0
        node = head
        while node is not None:
            sum_binary_digit = (sum_binary_digit << 1) | node.val
            node = node.next

        return sum_binary_digit
```
